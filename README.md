# Open Headers Browser Extension (Archived)

> **This repository is archived.** Development has moved to [OpenHeaders/open-headers-app](https://github.com/OpenHeaders/open-headers-app), which contains both the desktop app and the browser extension as a monorepo.

This repo remains as a read-only release host. Extension builds are published here automatically from the main repository.

## Install

- [Chrome Web Store](https://chromewebstore.google.com/detail/ablaikadpbfblkmhpmbbnbbfjoibeejb)
- [Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/open-headers/gnbibobkkddlflknjkgcmokdlpddegpo)
- [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/open-headers/)
